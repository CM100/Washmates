    getAll Gets all entries from a table that passed as parameter. Can be passed different query criteria.

    (
    • table
    • include
    • criteria
    )

    Defined in cloud\main.js:20

    Parameters:

    • table String

      The name of the table from which the results will be returned.

    • include Array

      An array of tables to include. e.g. javascript ['user', 'deliveryAddress']

    • criteria Object

      The criteria for restricting the query to certain results. Criteria should have as key the column name on which restrictions will be applied and as value another object with compare function name as key and compare value as value . Compare function name is a parse query comparisson function while value the value to compare the column against. e.g. javascript criteria = { objectId: {equalTo: '142sfa1'} }
